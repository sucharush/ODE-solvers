Newton's method.

This method implements Newton's algorithm for finding the zero of the following function: g(x) = x - yn - h*f(x, t_{n+1}), which is equivalent to solving y_{n+1} = y_n + h*f(y_{n+1}, t_{n+1}).

Parameters
ynthe current approximation of y(t_n)
tn1the time variable t_{n+1} = t0 + n*h
